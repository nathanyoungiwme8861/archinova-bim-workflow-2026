## Overview

ArchiNova Suite combines BIM workflows, parametric modeling, and automation tools into a single setup for architectural teams that need to move efficiently from concept development to documentation. Built for Windows, it is intended to help users keep design intent, analysis, and collaboration aligned within one working process.

The suite is well suited to projects where accuracy, data exchange, and iterative refinement are important. With IFC collaboration, structural analysis, and AI-assisted design integration, it offers designers and technical teams a practical space to test ideas, adjust geometry, and coordinate project data.

## Usage

A typical workflow with ArchiNova Suite looks like this:

1. Create or open a BIM project.
2. Build geometry with parametric controls and constraints.
3. Exchange data through IFC 4x3 when working with external collaborators.
4. Review structural feedback before finalizing changes.
5. Use the rendering pipeline to generate presentation-quality views.
6. Sync project updates through cloud collaboration when working in a team.

Example launch pattern:

`ArchiNovaSuite.exe`

If your distribution uses a project launcher, start it from the main application directory and load your design file from the home screen or project browser.
